In this Report I have used the below Tech stacks for the help of creating a Dashboards.


1.SQL
2.Power BI Desktop
3.Excel
4.Dax

6.Created a Project

1.Net profit
2.Net sales
3.Gross Margin
4.COGC - cost of goods sold
5.Post-Invoice deductions
6.Pre-invoice deductions
7.Gross price
8.YTG - Year to Go
9.YTD - Year to Date
